# Week 7: Statistical Business Analysis

## Overview
This project performs statistical analysis on sales data including descriptive statistics, hypothesis testing, correlation analysis, regression, and confidence intervals.

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- SciPy
- Scikit-learn

## Key Analysis
- Descriptive statistics
- Sales distribution analysis
- Correlation & heatmap
- Hypothesis testing
- Regression analysis
- Confidence intervals

## How to Run
1. Install dependencies from requirements.txt
2. Open statistical_analysis.ipynb
3. Run all cells

Screenshots of outputs and visualizations are included in the repository.
