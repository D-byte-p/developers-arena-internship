# Week 6 – Data Visualization

## Project Overview
This project focuses on visualizing sales data using Python, Matplotlib, and Seaborn to gain insights through charts.

## Visualizations Created
1. Bar Chart – Average price by category
2. Line Chart – Price trend over entries
3. Box Plot – Price distribution
4. Heatmap – Correlation between numerical columns
5. Pie Chart – Category distribution
